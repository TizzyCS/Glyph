# Glyph

**Glyph** is a minimalist programming language with a clean syntax and `.glf` files.  
Designed for fast prototyping and logical programming.

---

## Features

- Minimal syntax: variables, arithmetic, print statements  
- Simple conditionals: `if` / `else`  
- Tiny loops  
- Fast and easy to run  

---

## Example Glyph Program

```glf
# Hello World
name = "Bob"
print("Hello " + name)

x = 5 + 3
if x > 7:
    print("x is big!")
```

---

## Getting Started

### Run Glyph Programs

1. Clone the repo:

```bash
git clone https://github.com/YOUR_USERNAME/glyph.git
cd glyph
```

2. Run a `.glf` file:

```bash
python glyph_interpreter.py hello.glf
```

3. Optional: interactive REPL mode:

```bash
python glyph_interpreter.py --repl
```
